## About

Backend developer building **production-grade systems** in Python/Django and .NET — with a focus on industrial process automation, audit traceability, and clean, layered architectures.

I've worked in environments where precision isn't optional: systems that log every state change, integrate hardware, and run in containerized Linux deployments. That context shapes how I design software at every layer — from schema design to deployment.
